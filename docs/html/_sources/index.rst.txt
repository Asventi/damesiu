Bienvenue sur la documentation du projet de jeu Damesiu !
=========================================================
**Damesiu** (/dame'siou/) est un jeu de dames en python realise dans le cadre du
projet de programmation de premiere annee en BCSIU a l'ECAM LaSalle.
Vous pouvez retrouver ici le `Repo GitHub <https://github.com/Asventi/damesiu/>`_
