Lancement

Pour lancer le jeu, verifiez que vous avec bien Pyton 3.11 sur votre machine avec :

$ python --version
ou
$ python3 --version

Ensuite ouvrez un terminal a l’emplacement du jeu et lancez le jeu avec :

$ python run.py
ou
$ python3 run.py

Avertissement

Si vous etes sur windows, n’utilisez pas le terminal car il reagit mal avec curses depuis windows 10. Utilisez le powershell de windows.
