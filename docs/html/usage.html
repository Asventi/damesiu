Utilisation

Installation

Pour utiliser Damesiu, installez d’abord les dependances requises pour le jeu :

$ pip install requirements.txt

Important : Damesiu utilise la librairie native curses de python pour gerer l’affichage du jeu. Cette librairie n’est pas disponible sur Windows. Si vous etes sur windows vous devez installer la librairie pour windows :

$ pip install windows-curses

Lancement

Pour lancer le jeu, verifiez que vous avec bien Pyton 3.11 sur votre machine avec :

$ python --version
ou
$ python3 --version

Ensuite ouvrez un terminal a l’emplacement du jeu et lancez le jeu avec :

$ python run.py
ou
$ python3 run.py

Important : Si vous etes sur windows, n’utilisez pas le terminal car il reagit mal avec curses depuis windows 10. Utilisez le powershell de windows.

Comment jouer

Pour jouer, utilisez les fleches directionnelles pour vous deplacer et entree pour selectionner un pion, et selectionnez ensuite une case jouable representee par les ° vert, si vous pouvez sauter plusieurs pions vous devrez rejouer.
